# 3D Euler Path Finder

Interactive 3D visualization tool for discovering and rendering Euler paths on complex graph structures.

## Features

- Real-time 3D graph rendering with WebGL
- Euler path discovery using Hierholzer's algorithm
- Interactive node and edge manipulation
- Exportable path visualizations
- Support for directed and undirected graphs

## Tech Stack

Python, WebGL, Three.js, NumPy, NetworkX

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/3D-Euler-Path-Finder.git
cd 3D-Euler-Path-Finder
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
